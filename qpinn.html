About this Project

This project utilizes Pennylane, a quantum machine learning library, to implement Quantum Physics-Informed Neural Networks (QPINN) for solving the 3D heat diffusion equation. The solution involves advanced quantum circuits for encoding and processing the inputs to enhance the prediction accuracy compared to classical PINNs.

Technologies Used

  • Pennylane (Quantum Machine Learning)
  • PyTorch (Neural Network backend)
  • Python, NumPy, Matplotlib
  • High-performance computing (CUDA)
